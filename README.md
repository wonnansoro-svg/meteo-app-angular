# 🌦️ Application Météo Angular

Une application web interactive développée avec Angular qui permet de consulter les prévisions météorologiques en temps réel pour les villes du Sénégal et du monde entier.

## ✨ Fonctionnalités
* **Météo en temps réel :** Affiche la température, l'humidité, la vitesse du vent et une description détaillée.
* **Prévisions à 5 jours :** Visualisation claire des tendances météorologiques pour les jours à venir.
* **Recherche dynamique :** Barre de recherche intuitive pour interroger n'importe quelle ville.
* **Gestion des erreurs :** Alertes claires si la ville est introuvable ou en cas de problème de réseau.
* **Design Responsive :** Interface utilisateur moderne et attrayante conçue avec Bootstrap.

## 🛠️ Technologies Utilisées
* **Framework :** Angular 
* **Style :** CSS & Bootstrap 5
* **API :** OpenWeatherMap (Current Weather Data & 5 Day / 3 Hour Forecast)
* **Déploiement :** Vercel

## 🚀 Installation et Utilisation en local

### Prérequis
* Node.js installé sur votre machine.
* Angular CLI installé globalement (`npm install -g @angular/cli`).
* Une clé API gratuite sur [OpenWeatherMap](https://openweathermap.org/).
